Grailed Plus Privacy Policy

Last updated: March 16, 2026

Overview

Grailed Plus runs inside your browser to enhance Grailed listing pages. It stores your settings locally and sends only the remote requests needed to fetch exchange rates and Depop comparison data.

Data We Use

  • Grailed listing title, price, metadata, and thumbnail context visible in the current tab
  • User settings such as dark mode, currency, and market compare preferences
  • Depop search results and Depop thumbnail responses used to rank similar listings
  • Frankfurter exchange-rate responses used for optional currency conversion

Remote Requests

  • `https://*.grailed.com/*` for the page you are actively viewing
  • `https://*.depop.com/*` for Depop comparison searches and thumbnail fetches
  • `https://api.frankfurter.app/latest*` for exchange-rate data

Storage

Settings are stored in browser extension storage on your device. Grailed Plus does not run its own backend and does not upload your settings to a Grailed Plus server.

Data Sharing

Grailed Plus does not sell personal data. Data visible on the current Grailed listing page may be transformed into Depop search queries so the extension can fetch similar listings.

Code Execution

Grailed Plus does not execute remote code. Remote responses are treated as data only and are parsed locally inside the extension.
