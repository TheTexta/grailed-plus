Grailed Plus Offboarding

Disable or remove the local beta cleanly.

Disable / Remove

  1. Open chrome://extensions.
  2. Find Grailed Plus.
  3. Toggle it off to disable, or click Remove to uninstall.

Cleanup Debug State

If debug mode was enabled, clear it from DevTools Console:

localStorage.removeItem("grailed-plus:debug");

Known Beta Notes

If panel rendering stops after a major Grailed UI update, reload the page first. If still broken, capture console logs and listing URL for triage.
