Grailed Plus Privacy Policy

Last updated: March 17, 2026

Overview

Grailed Plus runs inside your browser to enhance Grailed listing pages. It stores your settings locally and sends only the remote requests needed to fetch exchange rates and Depop comparison data. Market Compare image similarity is computed locally in the extension with bundled MobileCLIP-S1 and ONNX Runtime assets.

Data We Use

  • Grailed listing title, price, metadata, and listing image context visible in the current tab
  • User settings such as dark mode, currency, and market compare preferences
  • Depop search results and Depop thumbnail responses used to rank similar listings
  • Local image embeddings and thumbnail fingerprints derived in memory to score market compare results
  • Frankfurter exchange-rate responses used for optional currency conversion

Remote Requests

  • `https://*.grailed.com/*` for the page you are actively viewing
  • `https://*.depop.com/*` for Depop comparison searches and thumbnail fetches, without requiring per-candidate listing-page fetches for similarity scoring
  • `https://api.frankfurter.app/latest*` for exchange-rate data

Storage

Settings are stored in browser extension storage on your device. Grailed Plus does not run its own backend and does not upload your settings to a Grailed Plus server. Model assets are packaged with the extension and loaded from the extension bundle only.

Data Sharing

Grailed Plus does not sell personal data. Data visible on the current Grailed listing page may be transformed into Depop search queries so the extension can fetch similar listings. Depop thumbnails are compared locally and are not sent to a Grailed Plus service for ML processing.

Code Execution

Grailed Plus does not execute remote code. Remote responses are treated as data only and are parsed locally inside the extension. The MobileCLIP-S1 model and ONNX Runtime Web files are bundled with the extension package; they are not downloaded from a CDN at runtime.
