# Store Listing Material (MV3 Beta)

## Title
Grailed Plus

## Short Summary
Price history and listing insights for Grailed listing pages.

## Description
Grailed Plus restores listing analytics directly on modern Grailed listing pages.

Current beta feature set:
- Price history
- Average price drop
- Next expected drop estimate
- Seller account creation date
- Listing metadata viewer

Technical notes:
- Manifest V3
- Chrome-first sideload beta
- Uses page bootstrap listing data (`__NEXT_DATA__`)
- SPA navigation-safe rendering
