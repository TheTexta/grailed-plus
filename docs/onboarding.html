Grailed Plus Onboarding

Get the MV3 beta running locally in Chrome.

Install Steps

  1. Open chrome://extensions.
  2. Enable Developer mode.
  3. Click Load unpacked.
  4. Select the repository src/ folder.
  5. Open any https://www.grailed.com/listings/... page.

Smoke Test Checklist

  • Grailed Plus panel appears near listing price/info area
  • Price history + avg drop metrics render
  • Expected drop text renders (or graceful fallback)
  • Metadata button opens listing JSON in a new tab
  • No duplicate panel when navigating between listings

Debug Toggle

Enable debug logs from DevTools Console:

localStorage.setItem("grailed-plus:debug", "1");
location.reload();
